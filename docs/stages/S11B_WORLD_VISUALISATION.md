# S11B — World Debug Layer Visualization MVP

## Цель этапа

Цель: поверх сетки показать debug-визуализацию слоев WorldGrid.


## Требования к реализации

На данном этапе требуется реализовать минимальную отрисовку всех существующих типов объектов:
- solid cells как colored rectangles;
- gas cells как tint/rect по total_particles;
- structures как outline/rect по occupied area, если структуры уже есть;

При создании мира добавить в него некоторое количество разных объектов из существующих реестров (газы, твердые клетки, структуры) и зафиксировать в коде, что это временное решение.

Важно учитывать, что текущий способ отрисовки - временный. Данный этап лишь требует создать базовый пайплайн для отрисовки данных мира. При этом отрисовка каждого отдельного типа объектов должна быть инкапсулированной, чтобы в дальнейшем можно было изменить эту отрисовку без необходимости лезть в другой код.

- видна сетка;
- видны solid cells;
- видны gas cells;
- видны структуры;
